1. コンピュータウイルスとは？

そもそも、コンピュータウイルスとは何を指すのでしょうか。
独立行政法人情報処理推進機構（IPA）によるコンピュータウイルスの定義[1]は、以下のようになっています。

『第三者のプログラムやデータベースに対して意図的に何らかの被害を及ぼすように作られたプログラムであり、次の機能を一つ以上有するもの』
（１）自己伝染機能
自らの機能によって他のプログラムに自らをコピーし又はシステム機能を利用して自らを他のシステムにコピーすることにより、他のシステムに伝染する機能
（２）潜伏機能
発病するための特定時刻、一定時間、処理回数等の条件を記憶させて、発病するまで症状を出さない機能
（３）発病機能
プログラム、データ等のファイルの破壊を行ったり、設計者の意図しない動作をする等の機能

出典
[1] IPA：「コンピュータウイルス対策基準」


2．ウイルスの主な種類

次に、コンピュータウイルスにはどのようなものがあるのかについて解説します。
コンピュータウイルスの主な種類としては、次の２つがあります。

  ①ファイル感染型
  ②マクロ感染型

 また、その他のマルウェア（悪意のあるソフトウェア）として、

  ③ワーム

 があります。
次項から、それぞれについてより詳しく解説します。


2-1．ファイル感染型

このタイプのウイルスは、拡張子が.COM、.EXEなどの実行型ファイルに感染して、ファイルの中にあるプログラムを書き換えることで増殖します。また、ファイル感染型ウイルスには「上書き型」「追記型」があります。「上書き型」はファイルを上書きするタイプで、「追記型」は元のファイルの一部として寄生したり、ファイルの未使用の領域に不正なコードを書き込むタイプです。[2]

具体例：PE_EXPIRO
このウイルスは、2010年に検知されたファイル感染型ウイルスです。しかし、他の典型的なファイル感染型ウイルスとは異なり、このウイルスは「情報収集機能」を備えているという特徴があります。[2][3]


2-2．マクロ感染型

このタイプのウイルスは、表計算ソフトや文書作成ソフトに付属するマクロ機能を利用して感染します。主に、不正な操作を行うマクロが埋め込まれた表計算ソフトなどのファイルが電子メールに添付されて送信され、ユーザはそれを開く、または開いてマクロを実行することでウイルスに感染し、コンピュータに不正な操作がなされます。[2]

具体例：Melissa（メリッサ）
このウイルスは、1999年に拡散が始まったマクロ感染型ウイルスです。当時は文書を開くとともにマクロが実行されるように設定されており、また当時としては新しい手口であったため、感染拡大が大きくなりました。[4]


2-3．ワーム（マルウェア）

ワームは、自分以外のプログラムに感染せずに自己増殖するという特徴をもつマルウェアです。感染するための宿主となるファイルを必要とせず、単独のプログラムとして活動するため、感染力が強い事でも知られています。[2][5]

具体例：LOVELETTER（ラブレター）
LOVELETTERは、2000年に猛威を振るったワームです。「I Love you」という件名のメールにワームが添付されて拡散したことから、この名前が付いたとされています。感染すると、メールに自己複製したワームを添付してOutlookの連絡先にある宛先に送信することで受信者に知り合いからのメールだと思わせ、更なる感染拡大を図ります。[6]

具体例：WannaCry（ワナクライ）
WannaCryは2017年に世界で同時発生し、ワーム的な手法を用いたランサムウェアとして話題になりました。ファイル共有のためのプロトコルである「SMBプロトコル」の脆弱性を悪用し、ワームを拡散させることで感染拡大を引き起こしました。[6]

3．ウイルスの感染経路

前項では、コンピュータウイルスの主な種類とその具体例を解説しました。ここではそれらを踏まえた上で、コンピュータウイルスの具体的な感染経路と、その対策について解説します。


3-1．ウイルスの感染経路

①ホームページの閲覧
現在のWebブラウザは、ホームページ上でさまざまな処理を実現させるため各種のプログラムを実行できるようになっていますが、これらのプログラムの脆弱性を悪用するウイルスが埋め込まれたホームページを閲覧すると、それだけでコンピュータがウイルスに感染してしまう危険があります。

②信頼できないサイトで配布されたプログラムのインストール
「あなたのコンピュータはウイルスに感染しています」のようなメッセージを表示し、あたかも無料のウイルス対策ソフトのように見せかけて、偽のウイルス対策ソフトを配布するwebサイトに誘導し、悪意のあるプログラムをインストールさせようとする手口があります。

③ネットワークのファイル共有
ウイルスによっては、感染したコンピュータに接続されているファイル共有ディスクを見つけ出し、特定のファイル形式など、ある条件で探し出したファイルに感染していくタイプのものがあります。このようなウイルスは組織内のネットワークを通じて、他のコンピュータやサーバにも侵入して感染を拡げる可能性があります。

④ファイル共有ソフトによる感染
ファイル共有ソフト（※１）では、不特定多数の利用者が自由にファイルを公開することができるため、別のファイルに偽装するなどの方法で、いつの間にかウイルスを実行させられてしまいます。

※１．ファイル共有ソフト… インターネットを利用して他人とファイルをやり取りするソフトウェアのこと。ファイル交換ソフトとも呼ばれている

⑤電子メールの添付ファイル
最近は、ファイル名を巧妙に偽装し、文書形式のファイルに見せかけて悪意のあるプログラムを実行させ、ウイルスに感染させる事例もあります。また、文書形式のファイルであっても、文書を閲覧するソフトウェアの脆弱性を狙った攻撃も増加しています。

⑥電子メールの HTML スクリプト
添付ファイルが付いていなくても、HTML形式で書かれているメールの場合、ウイルスに感染することがあります。HTMLメールはホームページと同様に、メッセージの中にスクリプトと呼ばれるプログラムを挿入することが可能なため、スクリプトの形でウイルスを侵入させておくことが出来ます。電子メールソフトによっては、HTMLメールのスクリプトを自動的に実行する設定になっているものがあり、その場合には電子メールをプレビューしただけでウイルスに感染してしまいます。

⑦USBメモリからの感染
多くのコンピュータでは、USBメモリをコンピュータに差し込んだだけで自動的にプログラムが実行される仕組みが用意されています。この仕組みを悪用して、コンピュータに感染するウイルスがあります。また、感染したコンピュータに後から差し込まれた別のUSBメモリに感染するなどの方法で被害を拡大させるものもあります。

⑧マクロプログラムの実行
Officeアプリケーションでは、マクロを作成する際に、高度なプログラム開発言語であるVBA（※２）を使用できるため、ファイルの書き換えや削除など、コンピュータを自在に操ることが可能です。そのため、マクロウイルスに感染した文書ファイルを開いただけで、VBAで記述されたウイルスが実行されて、自己増殖することがあります。

※２．VBA… Visual Basic for Applications


3-2. ウイルス感染対策

①怪しいサイトを閲覧しない
スパムメール同様、違法ダウンロードサイトなど不審なサイトを介して悪質なプログラムがばらまかれるケースが多いです。このようなサイトは閲覧しないようにしましょう。

②メールの URL リンクや添付ファイルを不用意に開かない
多くのウイルスはスパムメールによって拡散されます。不審なメールが届いた際は、開いて添付ファイルを開かないのはもちろん、閲覧自体を行わないようにしましょう。

③所有者や中身に覚えのないUSBメモリなどは使わない
コンピュータウイルスの感染には、USBメモリ、CD/DVDなどを介したものもあります。USBメモリなどを利用する場合には、事前にウイルスチェックをしてから利用しましょう。

④OSやソフトウェアを最新の状態にする
OSやソフトウェアのアップデートには、ウイルスなどの危険が入り込む可能性があるバグや不具合、設計上のミスといったプログラムの脆弱性の改善が含まれます。そのため、OSソフトウェアの更新がある場合はなるべく早く適用しましょう。

⑤セキュリティソフトを導入する
セキュリティソフトを使うことで、不正サイトへのアクセス、マルウェアや不正アプリの感染リスクを下げることができます。しかし、セキュリティソフトはパソコンの中のファイルやプログラムを監視するため、他のセキュリティソフトと競合して不具合を起こさないようにどちらかの動作を止めてしまうことがあるので、１つに決めて利用しましょう。
※セキュリティソフトについては、次項でより詳しく解説しています。


3-3. セキュリティソフトの重要性

ここで、セキュリティソフトには具体的にどのような役割があるのかを解説します。

①ウイルス感染を防ぐ
セキュリティソフトは、メールに添付されたファイルやWebサイトなど様々な経路でコンピュータに感染し危害を加えるウイルスを事前に察知し、ユーザーに警告、そして駆除することができます。これにより、ウイルスに感染してコンピュータの動作が遅くなったり、ファイルが破壊されてしまうなどといった危害からコンピュータを守ります。

②コンピュータ内の情報漏洩を防ぐ
セキュリティソフトは、コンピュータ内にある情報が外部に漏洩しないように常に監視することができます。使っているコンピュータが会社のものである場合は、個人情報だけでなく、社内の情報や取引先など外部の重要な情報の漏洩も防ぐことができます。

③迷惑メールの振り分け
セキュリティソフトは、見知らぬサイトから届く広告・勧誘の迷惑メールやウイルスを含んだファイル付きのメールなどの悪質なメールを、安全なメールと分けることができます。これにより、誤って迷惑メールを開いてしまう可能性が低くなり、より安全にメールの確認を行えるようになります。

4-3．定期的にバックアップをとる

[7]から、80%のユーザが定期的にバックアップを行っていない事実が明らかになりました。定期的にバックアップを行っていないと、コンピュータウイルスの被害に遭った際に思い出の写真やレポートなどの大事なデータをすべて失ってしまう可能性があります。
また、データそのものが個人の所有物のみであれば自己責任で済むものの、業務関係のファイルが破損してしまうと多大な損害を出してしまう恐れがあります。
思わぬトラブルを防ぐためには、バックアップを定期的にとる必要があります。しかし、バックアップを手動でとる習慣をつけるのは面倒なことと思います。そこで、我々は自動バックアップを利用することを推奨します。ここでは一例を紹介します。

Windows10 自動バックアップの設定の方法

1. バックアップ用のメディアをパソコンに接続し、スタートボタン右の検索ボックスに「更新」と入力し、検索結果に表示された「更新プログラムの確認」をクリック
2. 左のリストから「バックアップ」を選び、右に表示された「その他のオプション」をクリック
3. 「詳細設定の表示」をクリック
4. 「ドライブの選択」をクリックして、コピー先の外部メディアを選ぶ
5. 「詳細設定」をクリックして、ファイルのコピーを保存する頻度を決める。

4-4．パスワードを適切に設定し、管理する

総務省[8]によると、パスワード作成には以下の５点が重要とのことです。
(1) 名前などの個人情報からは推測できないこと
 (2) 英単語などをそのまま使用していないこと
 (3) アルファベットと数字が混在していること
 (4) 適切な長さの文字列であること
 (5) 類推しやすい並び方やその安易な組合せにしないこと

 しかし、５点のポイントをすべてカバーしたパスワードを考えることはとても面倒です。そこで、比較的手軽にパスワードを生成することができるサービス「パスワードジェネレータ」を紹介します。
例えば、このサイトのパスワードジェネレータは、「文字」「文字数」「個数」「構成」「乱数生成アルゴリズム」をそれぞれ選択できます。パスワードジェネレータを利用することで、設定の仕方やパスワードの選び方次第でこの５点のポイントすべてをカバーすることができます。

4-5．メールやショートメッセージ（SMS）、SNSでの不審なファイルやURLに注意する

サイバー攻撃の中には、メールに悪意のあるプログラムやフィッシングサイトのURLを添付して拡散するものがあります。言葉がおかしかったり心当たりがない内容などで、少しでも不審さを感じたら、絶対に開かないようにしてください。ここでは、初見では回避しにくい巧妙な偽造添付ファイルの例を紹介します。

偽造ファイルの例
・２重拡張子 「file.xlsx.exe」
・RLO表示 「filetxt.exe」→「fileexe.txt」に偽造
（RLO拡張子偽装とは、OSの文字設定表示を変更して拡張子を偽装し、ユーザに悪意のプログラムを実行させる攻撃手法。）

対策
巧妙に作られている偽造ファイルは、見分けることが困難です。不用意に添付ファイルを開かない、リンクをクリックしないことが最も有効な手段です。
マクロが自動で有効になるような設定は行わないことも重要です。
※注意：エクスプローラーで表示されるアイコンは偽造可能
たとえば、拡張子.exeのファイルをpdfのアイコンで表示させることが可能です。
つまり、pdfファイルを開くつもりでクリックしたのに、実行ファイルが実行されてしまうと言うことが起こりえます。アイコンだけでなく、必ずファイルの拡張子まで確認しましょう。
※圧縮ファイルも「ファイル詳細」で見ればアイコン表示されずにチェック可能です。

不審なファイルを開いてしまった場合

1. 当該端末をネットワークから切り離す（LANケーブル抜線や無線LAN機能の無効化）
2. セキュリティソフトでウイルススキャンを実施する
3. 必要に応じて、スキャン結果や開いてしまったファイルの情報、セキュリティソフトの定義ファイルの情報等をまとめてセキュリティベンダに相談する
4. 通信ログを確認（監視）する
5. ウイルス感染が確認できた場合は当該端末の初期化または保全を検討する
（セキュリティソフトによる駆除ができた場合でも、感染端末に未知のウイルスが残されている可能性がゼロとは言えないため、安全な利用のためには初期化が推奨される。また、ウイルス解析のためには感染したままの状態で保全することが望ましい場合も。）
※ウイルスや不正アクセスによる被害が疑われ、相談の必要がある場合には情報セキュリティ安心相談窓口に相談しましょう。

4-6．偽のセキュリティ警告に注意する

偽警告とは？
偽警告とはWebサイトを閲覧中に突然表示される偽物の警告のことです。
誘導にのってしまうと悪意のあるWebサイトへのアクセスや悪意のあるプログラムのダウンロードをさせられることがあります。種類が多く、タイミングによっては判断が難しい場合があるので、日頃から十分に注意しておく必要があります。
次のような対策を覚えておきましょう。

偽警告の対策
１. 無視する
広告が表示されてもメッセージに従わず、警告が表示されているブラウザのタブを閉じたり、ブラウザ自体を終了させたりするのが最も効果的です。偽警告が表示されている環境によっては、ブラウザがフリーズしてしまい操作できなくなるケースもあります。このような場合は、Windowsのタスクマネージャーを使って、ブラウザ自体を強制終了させましょう。
２. URLを確認する
誘導先のWebサイトがSSL（※３）に対応しているか、対応していないかが正規サイトかそうでないかの判別に利用できます。
３. ブラウザのキャッシュを削除する
ブラウザには閲覧記録やWebサイトのデータがキャッシュとして保存される。もし偽警告が表示されてしまった場合、再度表示されることを防ぐために、ブラウザの閲覧記録やキャッシュファイルを削除しておきましょう。

※３．SSL… Secure Sockets Layer．インターネット上におけるウェブブラウザとウェブサーバ間でのデータの通信を暗号化し、送受信させる仕組み。
